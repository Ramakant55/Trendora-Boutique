# Luxury Boutique Website

A premium, modern, and luxurious boutique website UI built with React, Vite, and Tailwind CSS.

## Features

- Elegant, feminine, and classy design inspired by high-end fashion brands
- Soft pastel color palette with gold accents
- Fully responsive for mobile, tablet, and desktop
- Smooth animations using Framer Motion
- Glassmorphism effects
- Modern typography with Poppins and Playfair Display fonts

## Sections

1. Navigation Bar with glassmorphism effect
2. Hero Section with full-width banner
3. Featured Collections with hover animations
4. Product Grid with rating stars and "Add to Cart" buttons
5. About Boutique section with interior image
6. Customer Testimonials
7. Offer Banner with gradient background
8. Comprehensive Footer with contact info and newsletter signup

## Technologies Used

- React
- Vite
- Tailwind CSS
- Framer Motion for animations

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm run dev
   ```
4. Open your browser to http://localhost:5173

## Customization

- Colors can be modified in `tailwind.config.js`
- Fonts can be changed in `index.css`
- Content can be updated in each component file
